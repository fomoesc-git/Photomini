# Photomini 图片压缩工具

> 极简图片压缩与格式转换工具

## 功能特性

- 🖼️ **图片压缩** - 支持 JPG/PNG，拖入即压缩
- 🔄 **格式转换** - 支持 JPG ↔ WebP 双向转换
- 📊 **质量可调** - 10%-100% 质量滑块
- 🎯 **傻瓜操作** - 拖入文件自动处理，无需确认
- 📦 **极简设计** - 不生成任何日志或临时文件

## 技术栈

- **前端**: React 18 + TypeScript + Vite
- **桌面框架**: Tauri 2.x
- **后端**: Rust
- **图片处理**: image-rs

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建发布
npm run tauri build
```

## 系统要求

- macOS 11.0+
- Windows 10+（后续支持）

## 使用说明

1. 选择功能 TAB（图片压缩 / Webp转换）
2. 设置压缩/转换质量
3. 选择输出方式（覆盖原文件 / 另存至）
4. 拖入图片文件
5. 自动处理，显示结果
6. 点击清空按钮重置
