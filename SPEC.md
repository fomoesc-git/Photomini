# Photomini 图片压缩工具 - 产品规格书

## 1. 项目概述

- **项目名称**: Photomini（随手压）
- **项目类型**: 桌面工具软件
- **核心功能**: 极简图片压缩与格式转换工具
- **目标用户**: 需要快速压缩图片、转换格式的非专业用户
- **支持平台**: macOS（开发版）、Windows（后续）

## 3. 功能规格

### 3.1 TAB1 - 图片压缩

| 功能 | 描述 |
|------|------|
| 输入格式 | JPG/JPEG、PNG |
| 压缩算法 | 基于质量因子的有损压缩 |
| 质量范围 | 10% ~ 100%，默认 75% |
| 输出格式 | 与输入格式一致 |
| 输出选项 | 覆盖原文件 / 另存至指定文件夹 |

### 3.2 TAB2 - Webp转换

| 功能 | 描述 |
|------|------|
| 模式 | JPG → WebP / WebP → JPG（互斥按钮） |
| 转换质量 | 10% ~ 100%，默认 75% |
| 输出选项 | 覆盖原文件 / 另存至指定文件夹 |

### 3.3 文件列表显示

| 列 | 内容 |
|----|------|
| 文件名 | 原始文件名 |
| 原大小 | 文件原始大小，格式：KB/MB |
| 箭头 | → |
| 新大小 | 处理后大小，格式：KB/MB |
| 状态 | ✓ 完成 / ⟳ 处理中 / ✗ 失败 |

### 3.4 交互规格

| 交互 | 行为 |
|------|------|
| 拖入文件 | 立即自动开始处理，无需确认 |
| 点击清空按钮 | 清除列表，恢复空闲状态 |
| 窗口关闭 | 程序退出 |

### 3.5 约束条件

- **不生成任何日志文件**（.log / .debug 等）
- **不生成任何临时文件**（处理在内存中完成）
- **不支持文件夹拖入**（仅支持单个或多个文件）
- **不支持子文件夹递归**
